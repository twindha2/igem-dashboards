Save to Google Drive & share

5 minutes

Step-by-step

  1. Use the Download action in the toolbar and choose the wiki-framework/ folder — you'll get a .zip of all four HTML files plus this index.
  2. Go to drive.google.com → New → Folder. Name it UH iGEM 2026 — Wiki Framework.
  3. Unzip locally, then drag the four HTML files (plus this index) into the Drive folder.
  4. Right-click the folder → Share → add team emails, or set Anyone with the link (Viewer) for a shareable URL.
  5. In Drive, each .html file previews in the browser when clicked — no download needed for reviewers.

Tip: for a single link that opens the framework file directly, right-click 01-framework-no-style.html in Drive → Get link. That's the URL to paste into Slack.
